This is how the gallery looks like, it has a search bar, thumbnail size slider, and dropdown menus so you can filter your images by date, model, LoRa, Schedulers, Samplers.

The Keep/Trash buttons do not modify the image in any way, if you select an image and click on "keep", a folder named "Keepers" will be created and the image will be moved there, same happens when you click "Trash", you can then delete the trash folder contents later after you review your images. The gallery is kinda self explanatory, but feel free to ask about anything, or suggest edits.
